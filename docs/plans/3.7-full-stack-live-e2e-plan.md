## Performance Evaluation Contract

The full-stack run must include the same sandbox performance evidence as narrower scenarios.

Required assertions:

- Lowerdir disk stays O(1) across the whole run: max `workspace_tree_bytes == 0` and `workspace_tree_exists == 0` for namespace-mode resource snapshots.
- Tool latency report includes p50/p95/max for shell, read/write/edit, LSP, and plugin calls.
- Slow-call attribution separates provider/agent time from sandbox operation time.
- Synthetic verifier failures are expected workflow events; sandbox `internal_error`, stale lowerdir, missing layer, and mount failure are not.
- End-of-run cgroup metrics use run delta for I/O and CPU; lifetime counters are not reported as per-run usage.
