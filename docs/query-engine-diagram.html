EphemeralOS Query Engine

Agent Tool Call Loop Architecture (Control Flow)

User Prompt
    │
    ▼
┌─────────────────────────────┐
│      EphemeralAgent.run()    │
│  (agent wrapper, thin facade)  │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│      QueryEngine.submit_message()│
│  (created first by spawn_agent)│
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│      QueryContext           │
│  (api_client, tool_registry)│
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────────────────────┐
│           run_query() (query.py)             │
│  ┌─────────────────────────────────────┐   │
│  │  while active:                       │   │
│  │                                       │   │
│  │  ┌───────────────────────────────┐  │   │
│  │  │  api_client.stream_message()  │  │   │
│  │  │  (OpenAICompatibleClient)     │  │   │
│  │  └───────────────┬───────────────┘  │   │
│  │                  │                   │   │
│  │                  ▼                   │   │
│  │  ┌───────────────────────────────┐  │   │
│  │  │  ApiMessageCompleteEvent      │  │   │
│  │  │  - final_message              │  │   │
│  │  │  - usage                      │  │   │
│  │  │  - stop_reason                │  │   │
│  │  └───────────────┬───────────────┘  │   │
│  │                  │                   │   │
│  │                  ▼                   │   │
│  │  ┌───────────────────────────────┐  │   │
│  │  │  Has tool_uses?               │  │   │
│  │  │  ─────────────────────────    │  │   │
│  │  │  NO → yield AssistantTurn     │  │   │
│  │  │       Complete → RETURN       │  │   │
│  │  │  YES → continue               │  │   │
│  │  └───────────────┬───────────────┘  │   │
│  │                  │                   │   │
│  │                  ▼                   │   │
│  │  ┌───────────────────────────────┐  │   │
│  │  │  For each tool_call:          │  │   │
│  │  │  ┌─────────────────────────┐  │  │   │
│  │  │  │ 1. platform pre-hook      │  │  │   │
│  │  │  │    TOOL_USE (optional)    │  │  │   │
│  │  │  └────────────┬────────────┘  │  │  │   │
│  │              ▼              │  │  │   │
│  │  ┌─────────────────────────┐│  │  │   │
│  │  │ 2. tool_registry.get()    ││  │  │   │
│  │  │  │    (resolve tool)         ││  │  │   │
│  │  │  └────────────┬────────────┘│  │  │   │
│  │              ▼              │  │  │   │
│  │  ┌─────────────────────────┐│  │  │   │
│  │  │ 3. tool.execute()         ││  │  │   │
│  │  │  │    (BaseTool.execute)     ││  │  │   │
│  │  │  └────────────┬────────────┘│  │  │   │
│  │              ▼              │  │  │   │
│  │  ┌─────────────────────────┐│  │  │   │
│  │  │ 4. platform post-hook     ││  │  │   │
│  │  │  │    TOOL_USE (optional)    ││  │  │   │
│  │  │  └────────────┬────────────┘│  │  │   │
│  │              ▼              │  │  │   │
│  │  ┌─────────────────────────┐│  │  │   │
│  │  │ 5. ToolResultBlock        ││  │  │   │
│  │  │  │    (append to messages)   ││  │  │   │
│  │  │  └─────────────────────────┘│  │  │   │
│  │  └─────────────────────────────┘  │   │
│  │                                       │   │
│  │  Parallel execution if multiple tools │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
               │
               ▼
        StreamEvents:
        - AssistantTextDelta
        - ThinkingDelta  
        - ToolExecutionStarted
        - ToolExecutionCompleted
        - AssistantTurnComplete
