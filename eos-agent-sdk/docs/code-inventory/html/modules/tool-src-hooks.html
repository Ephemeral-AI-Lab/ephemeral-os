HookEngine class · line 66 · public
Module 
export class HookEngine

Callback dispatch over one run's merged entries (globals first, then per-agent). Pre/post entries matching the call run in parallel; any deny wins and reasons join. A throwing pre/post hook resolves as deny with the thrown message (fail-closed; a host wanting fail-open catches inside its callback). A throwing turnBoundary hook is recorded and skipped. A broken hook never wedges a batch.
