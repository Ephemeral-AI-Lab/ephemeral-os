toProviderError function · line 118 · public
Module 
export function toProviderError( error: unknown, phase: "open" | "stream", requestIdFallback?: string, ): ProviderError

Map an error thrown by an sdk call to the owned taxonomy. `phase` selects the fallback for errors with no http status: request construction failures at `open`, stream parse failures while iterating a `stream`. Abort errors never reach this mapping - providers rethrow them as-is when the caller's signal is aborted.
