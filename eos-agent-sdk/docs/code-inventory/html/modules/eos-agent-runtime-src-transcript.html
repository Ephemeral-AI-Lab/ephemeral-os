RunLog class · line 115 · public

export class RunLog

Per-run JSONL writer: one ordered append queue, fed by the runtime's event subscriber. `events.jsonl` records lifecycle facts, `transcript.jsonl` keeps the Phase 04.5 conversation artifact, and `result.jsonl` receives one terminal rollup line. A failed write latches and resurfaces on every later `flush()`.
