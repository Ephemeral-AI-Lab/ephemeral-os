A. Swim-Lane Timeline · per agent, per attempt

The attempt is laid out as horizontal time. Each agent (main, subs, helpers) gets its own lane; spawn arrows drop diagonally from parent to child. Tool calls render as colored blocks on the agent's lane. You see, at a glance, what was running when — and the now line shows live progress.

Inspired by Temporal Timeline View + Chrome DevTools Performance.

B. Hierarchical Trace · call tree + node detail

The attempt is a call tree: orchestrator → code-writer → edit_file. Select any node to see its input message, output message, latency, tokens, and metadata. A miniature DAG in the right pane keeps spatial context. Best when you ask "why did this attempt take that path?"

C. Live Agent Canvas · radial DAG with flowing edges

The attempt's agent DAG as a live canvas: nodes are agents, edges animate when messages flow. The active agent pulses; tools tick in the right pane. Cinematic and intuitive for stakeholders — feels like watching a control room map. Best for "is everything healthy right now?"

Inspired by AutoGen Studio canvas + n8n flow editor.

D. Mission Control · linked DAG + event log (recommended)

Strip of attempts across the top (compare iterations side-by-side). Body splits into DAG on the left (structure) and event log on the right (causality). Hovering a node highlights its events; clicking an event focuses the node. Footer pins persistent sandbox state. One screen answers "where" and "when" at the same time.

Inspired by Datadog APM + Sentry trace + Linear's issue layout.

Which to pick

Aspect A · Swim-Lane B · Hierarchical Trace C · Live Canvas D · Mission Control
Mental model Temporal time Call tree Animated control room Linked graph + log
Strongest at Parallelism, "what overlapped" Deep introspection of a span Live ops, demos, stakeholder view Daily driver across all needs
Weakest at Deep nested structure Real-time at a glance Dense data, replay scrubbing Width — needs 3-pane real estate
Live updates Now-line + lane block growth Tree nodes blink Edge flow animation All three
Best for replay ★★★ scrubbing ★★ jump-to-span ★ static feel ★★★ scrub + jump
Cost to build Medium (custom timeline) Low (off-the-shelf tree) High (force-graph + anim) Medium-High (composes A+B)
Recommendation: ship D · Mission Control as the default attempt view. Reuse the swim-lane (A) as the "Timeline" tab, the trace tree (B) as the right-pane drill-down, and the canvas (C) as the "Live demo / present" mode. Goal/iteration are navigation only — the attempt is the workspace.
