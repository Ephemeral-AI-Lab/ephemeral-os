# Sandbox Workspace Class Inventory

The maintained class inventory is the generated HTML artifact under
[`html/index.html`](./html/index.html).

Regenerate it from the live sandbox source with:

```bash
cargo run --manifest-path scripts/class-inventory/Cargo.toml
```

The old per-crate Markdown snapshots were retired because they were not
generated by the current tool and had drifted to deleted crate names and source
paths. The generated HTML inventory is the source of truth for current crate
symbols; the hand-curated contract docs live separately under
`sandbox/docs/contract/`.
