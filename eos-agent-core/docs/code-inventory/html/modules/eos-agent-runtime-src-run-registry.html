export class RunRegistry

One typed map of runs. The run facts live exactly once, in the state record; the registry adds only what the record must not hold - the live handle and the registry-level status. Terminal runs stay listed for the process lifetime (eviction is a deferred seam), so transcript reads against finished runs keep working.
