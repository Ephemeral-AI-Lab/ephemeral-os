export async function* retryStream( cfg: RetryConfig, attempt: () => AsyncIterable<LlmStreamEvent>, signal?: AbortSignal, ): AsyncGenerator<LlmStreamEvent, void, undefined>

Wrap a per-attempt stream factory in the visible-output retry gate. One `emittedVisible` flag spans all attempts: once any delta variant has been forwarded, a later failure fails fast - re-running the request would duplicate text and double-dispatch tool_use ids. No new attempt starts after `signal` aborts, and abort errors are rethrown as-is.
