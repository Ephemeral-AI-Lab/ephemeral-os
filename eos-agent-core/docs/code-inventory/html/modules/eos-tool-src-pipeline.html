bindTool function · line 57 · public
Module 
export function bindTool(definition: ToolDefinition, deps: BindToolDeps): BoundTool

Bind one definition to the per-call pipeline: abort check -> isolated guard -> parse -> PreToolUse -> execute -> PostToolUse(/Failure) -> stamping. Never throws; every path returns a result the executor can record. Timing brackets `execute()` only - pre-execution rejections stamp both times with the rejection instant, and slow hooks never masquerade as slow tools.
