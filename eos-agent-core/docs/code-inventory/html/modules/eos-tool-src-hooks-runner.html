export class HookEngine

Runs operator-configured hooks for one event: all matching hooks in `Promise.all`, each through its adapter, folded by the precedence kernel. Never throws - per-hook failures become passthrough warnings. Three distinct channels, never collapsed: structured decision (stdout JSON), model feedback (exit 2 stderr), operator warning (everything else).
