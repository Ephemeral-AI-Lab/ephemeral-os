AgentRunState interface · line 13 · public
Module 
export interface AgentRunState

The per-run metadata record, assembled once by the composition root. Data only - a port or service in here would be the rejected ambient `ToolRuntime`. Everything except `workspace.isIsolated` is readonly; that one cell's writers are the sandbox family's workspace-mode tools. Tools never receive this record at call time - `ToolCallMeta` nests its frozen `AgentRunSnapshot` instead.
